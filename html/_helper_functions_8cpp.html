◆ generateRandomIP()

string generateRandomIP ( )

Creates a random IP address for incoming/outgoing network requests.

Generates a random IPv4 address in the 192.168.1.x subnet.

  • Uses a standard 192.168.1.x prefix and generates a random final octet between 0 and 999 to simulate diverse network traffic.
  • Returns
    string The formatted IP address string.

◆ generateRequest()

Request * generateRequest ( char type)

Allocates and initializes a new Request object with random parameters.

Generates a new random network Request.

  • The required processing time is calculated as a random integer between the globally defined min_task_time and max_task_time.
  • Parameters
    typeThe required server type for this request.
    Returns
    Request* Pointer to the dynamically allocated Request.

◆ generateWebserverIP()

string generateWebserverIP ( )

Creates a unique, sequential IP address for internal Webservers.

Generates sequential IPv4 addresses for Webservers.

  • Automatically increments a global counter so no two webservers share an IP.
  • Returns
    string The formatted sequential IP address string.
