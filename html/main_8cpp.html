Detailed Description

The entry point for the load balancer simulation.

  • This file handles command-line argument parsing, initializes the simulation parameters, generates the simulated network traffic, and runs the main clock cycle loop to process requests through the Switch and LoadBalancers. It also tracks performance metrics and outputs simulation summaries.
