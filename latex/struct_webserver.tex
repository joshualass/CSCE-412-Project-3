\doxysubsection{Detailed Description}
Simulates a physical server executing network requests over time. 


\begin{DoxyItemize}
\item A \doxylink{struct_webserver}{Webserver} can be toggled on or off (active/\+inactive) by a \doxylink{struct_load_balancer}{Load\+Balancer}. When active and assigned a \doxylink{struct_request}{Request}, it counts down the required clock cycles until the job is fully processed. 
\end{DoxyItemize}
