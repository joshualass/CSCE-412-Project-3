\doxysubsection{Detailed Description}
The entry point for the load balancer simulation. 


\begin{DoxyItemize}
\item This file handles command-\/line argument parsing, initializes the simulation parameters, generates the simulated network traffic, and runs the main clock cycle loop to process requests through the \doxylink{struct_switch}{Switch} and Load\+Balancers. It also tracks performance metrics and outputs simulation summaries. 
\end{DoxyItemize}
