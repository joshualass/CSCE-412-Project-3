\doxysubsection{Detailed Description}
Acts as a router directing incoming requests to the appropriate \doxylink{struct_load_balancer}{Load\+Balancer}. 


\begin{DoxyItemize}
\item The \doxylink{struct_switch}{Switch} holds multiple Load\+Balancers, each dedicated to a specific type of network request. It inspects incoming requests and forwards them to the \doxylink{struct_load_balancer}{Load\+Balancer} matching the request\textquotesingle{}s type. 
\end{DoxyItemize}
